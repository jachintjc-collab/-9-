레지던트 이블 레퀴엠을 4시간 이내에 클리어하는 방법 — 스피드런 공략

한국어판 작성자 노트: 이 페이지는 단순한 영어 번역이 아닙니다. 서울 거주 RE 시리즈 23년 팬 Jachin이 PS5/Steam에서 RE9을 직접 6회 플레이 (Casual → Insanity → NG+ Insanity)한 후 작성. 한국 RE 커뮤니티에서 통용되는 영어 용어(Mortal Edge, NG+, Insanity 등)는 그대로 유지하면서 한국어 독자에게 적합한 표현으로 구성했습니다.

검증 버전: v1.02 · 마지막 검토: 2026년 5월 30일 · 영어 원본과 비교: 100% 내용 동등 + 한국어 컨텍스트 추가

2026년 5월 게시 · 12분 읽기 · 스피드런 전략
⚠ 스포일러 수위: 중간

레지던트 이블 레퀴엠의 완전한 스피드런 공략입니다. 목표 시간: 4시간 미만. 최적 루트, 필수 스킵, 무기 로드아웃, 보스 전략, 그리고 스피드러너 실버 트로피를 가장 쉽게 해금하는 방법을 다룹니다. 무한 레퀴엠을 활용한 인새니티 플레이와 결합하면 골드 트로피까지 한 번에 노릴 수 있습니다.

스피드런 셋업 요건

💡 최고의 스피드런 조합: 인새니티 + 무한 레퀴엠 장착으로 진행하세요. 이렇게 하면 스피드러너 트로피(5시간 미만)와 리마커블 에이전트 트로피(인새니티 클리어)를 한 번의 플레이로 동시에 달성합니다.

// 런 시작 전 해금

  1. 메인 스토리를 한 번 클리어 — 인새니티 + 스페셜 콘텐츠 스토어 해금
  2. 무한 레퀴엠 구매 스페셜 콘텐츠 스토어에서 (~6,000 CP)
  3. 선택: 무한 RPG-7 구매 (~8,000 CP) — 빅터 보조용
  4. 이전 런의 Mortal Edge Hatchet 보유 (NG+로 이월됨)

// 게임 내 설정

  • 옵션 메뉴에서 컷신 스킵 ON 설정
  • 자동 조준을 STRONG으로 설정 (귀중한 초 단위 절약)
  • 더 빠른 전환을 위해 조준 보조 토글 비활성화
  • 퀵턴을 편한 버튼에 바인딩

필수 스킵 (총 ~45분 절약)

// 6분 절약미스터 라쿤 25개 전부 스킵
신경 쓰지 마세요. 스피드런 루트는 티어 3 무기 업그레이드가 필요 없습니다. 동선 바로 위에 있지 않은 메모리엄은 전부 스킵하세요.
// 8분 절약5개 금고 중 4개 스킵 (작업실 금고만 진행)
작업실 금고(R60·L40·R80)는 라쿤 라운드업 지도를 줍니다 — 스피드런에는 쓸모없습니다. 모든 금고를 통째로 스킵하고; No Safe Is Safe 트로피는 포기하세요.
// 12분 절약선택 보스 스킵 (셰프 + 청크 #1)
싸우지 마세요. 셰프와 청크 #1은 완전히 스킵 가능합니다. Eye Spy 참은 좋지만 무한 레퀴엠이 있으면 필수가 아닙니다.
// 5분 절약파일 75개 전부 스킵
파일은 어떤 진행도 막지 않습니다. 모든 파일을 지나쳐 달리세요. "Case Closed!" 트로피는 수집런용이지 스피드런용이 아닙니다.
// 8분 절약정원 사이드 퀘스트 스킵
씨앗 심기 = 선택. 그린 썸 트로피는 스피드러너와 함께 스택되지 않습니다.
// 6분 절약룰렛 휠 이스터에그 스킵
RNG 의존적이고 골동 동전 1개만 절약합니다. 시간 투자 대비 가치가 없습니다.

구간별 루트

// 렌우드 (목표: 12:00)

선형 오프닝. 스킵 기회가 없습니다 — 그냥 튜토리얼을 최대한 빠르게 진행하세요. 대사 빠르게 넘기고, 튜토리얼 적을 신속히 패리한 뒤, 복도를 따라가세요.

  • 컷신을 즉시 스킵 (버튼 프롬프트)
  • 초반 탄약은 줍지 말 것 — 나중에 충분히 얻습니다
  • 좀비를 지나쳐 달리고; 교전하지 마세요

// 케어 센터 그레이스 (목표: 55:00)

가장 긴 구간. 쿼츠 퍼즐 3개를 빠르게 모두 풀기 — 필수입니다. 그 외에는 전부 스킵.

  1. 태양 쿼츠 (수석 연구원실): 별 · 태양 · 달 · 태양
  2. 달 쿼츠 (의장실): 달 · 태양 · 별 · 달
  3. 별 쿼츠 (VIP 스위트): 모래시계 해법 (보통 태양 · 별 · 달 · 별)

중요: 혈액 표본 1번 해결 (변성) — 용혈 주사기 확보를 위해. 나중에 2-3개가 필요합니다. 표본 2번과 3번은 스킵.

// 케어 센터 레온 (목표: 25:00)

다락방에서 MSBG 500을 즉시 픽업하세요. 무한 레퀴엠으로 다락방의 청크 #2를 원샷. 안뜰 출구로 곧장 달려가세요.

// 라쿤시티 (목표: 60:00)

여러 보스 전투:

  • 미스터 X 추격 — 싸우지 말고 루트만 달리세요 (미스터 X 공략 참조)
  • 슈퍼 타이런트 — 무한 레퀴엠으로 심장 사격. ~90초
  • 타이탄 스피너 — 무한 레퀴엠으로 엄폐-점사. ~3분

중요: 건샵 켄도의 저장고에서 시간을 낭비하지 마세요 — 무한 레퀴엠만 있으면 충분합니다.

// ARK 시설 (목표: 55:00)

플랜트 43 → 커맨더 → 복도 → 빅터:

  • 플랜트 43 — 캐니스터 트럭 폭발 피니셔 노리기 (페이즈 2 스킵)
  • 커맨더 — Mortal Edge 패리 사용; 인새니티 + 무한 레퀴엠으로 ~3분
  • ARK 복도 — 순수 달리기 통과. 전투 불필요.

// 빅터 기든 (목표: 15:00)

최종 보스. 무한 레퀴엠으로:

  1. 페이즈 1: 보라색 차지 결정 사격 → 2분
  2. 페이즈 2: 머리의 농포 파괴 → 코어 → 5-8분
  3. QTE 피니셔 프롬프트

4시간 미만 런용 로드아웃

최적 로드아웃:
주무기: 무한 레퀴엠 리볼버 (대부분의 적을 원샷)
보조: 무한 RPG-7 (긴급 보스 데미지)
근접: Mortal Edge Hatchet (패리 전용, 절대 공격하지 말 것)
참: Eye Spy Charm 획득 가능하면 (랜덤 치명타 세이브)

세이브 전략

스피드런은 세이브가 적을수록 유리합니다:

  • 주요 보스 전(미스터 X, 플랜트 43, 커맨더, 빅터) 직전에만 세이브
  • RNG 요소 전(모래시계 퍼즐)에 세이브
  • 탐색 중에는 세이브하지 말 것 — 각 세이브당 ~5초 손실
  • 같은 슬롯 사용 — 멀티 슬롯 세이브는 시간 낭비

흔한 시간 손실 실수

  • 파일을 읽으려고 멈추기 — 절대 금지. 전부 스킵 가능합니다.
  • 선택 좀비와 교전 — 지나쳐 달리세요. 한 발만 쏴도 2-3초 낭비.
  • 불필요한 탄약 줍기 — 무한 레퀴엠으로 의미 없습니다.
  • 실수로 사이드 보스 트리거 — 청크 #1과 셰프의 트리거 지점을 알아두세요.
  • 너무 자주 세이브 — 세이브 1회당 4-6초 추가.
  • 태양 퍼즐을 틀리게 풀기 — 답을 외우고, 단서를 다시 읽지 마세요.

최상위 기록 벤치마크

트로피 스택 전략

인새니티에서 효율적인 스피드런은 한 번의 플레이로 트로피 3개를 해금합니다:

  • "리마커블 에이전트" 골드 — 인새니티 클리어
  • "스피드러너" 실버 — 5시간 미만
  • "레퀴엠 컴플리트" 골드 — 스토리 클리어 (첫 런에서 이미 달성)

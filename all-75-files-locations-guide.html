How Files work

Files are document collectibles scattered across the game world. Some give you puzzle clues (like safe code hints), others fill in lore. They're tracked in the Files tab next to your inventory; but the in-game order doesn't always match the chronological order you find them in.

  • Reading is mandatory — picking a File up isn't enough; you must open and view it
  • Both Grace and Leon collect Files; the total of 75 spans both characters
  • Some Files are story-automatic, you can't miss them, they trigger via cutscene
  • Most Files are environmental — you have to actively look around rooms
  • Reward: Case Closed Silver trophy + 3,000 CP from the Challenge menu

Tips and missable warnings

// TOP MISSABLE FILES

Based on community feedback, these are the most commonly missed Files:

  • File #14 (Director's Notes); Requires the Pencil from a separate room. Skip this and you skip the safe code clue too.
  • File #32 (Pantry Inventory) — Locked behind The Chef boss fight. Skipping the Chef means losing this file forever.
  • Files #39-42 (Marie's Drawing + Subject 17 Reports), Critical for NG+ Final Puzzle lore. Don't rush past Collection & Testing.
  • File #57 (RPD Library Note); Easy to miss during the Mr. X chase. Return to the library after defeating Super Tyrant.
  • Files #70-73 (ARK Access Logs & Safes) — All locked behind the final ARK sequence. Once you trigger the Victor cutscene, they're gone.

// EFFICIENT COLLECTION ROUTE

For first-time 100% runs:

  1. Save before each area transition, use multiple slots
  2. Clean Wrenwood completely before the hotel cutscene
  3. Care Center: do 1F → 2F → Basement in that order, never the reverse
  4. Defeat Chef + Chunk #1 as Grace for missable files
  5. Return to RPD Library after Mr. X is defeated for File #57
  6. ARK: clean all files BEFORE entering the final Archive console

// CHALLENGE POINTS REWARD

✅ Reward summary:
• Case Closed Silver trophy unlocks on File #75
• 3,000 Challenge Points awarded immediately
• Use these CP at the Special Content Store for Infinite Ammo unlocks (Infinite Requiem = 6,000 CP, Infinite Carbine = 5,000 CP)
• Combined with all 25 Mr. Raccoons (+5,000 CP) and 49 trophies, you'll have enough CP for the full Insanity speedrun loadout

// IF YOU MISS A FILE

If you progress past a point of no return without all 75 files, your options are:

  • Reload a manual save from before the transition; best option if you saved often
  • Continue to credits and start NG+ — files reset, but you keep your weapon unlocks and Infinite Ammo
  • Use Insanity speedrun strategy, knowing all locations from this guide, a complete run takes ~4 hours with Infinite Requiem equipped

The Files menu shows you exactly which file numbers you're missing, making it easy to plan a targeted second playthrough.

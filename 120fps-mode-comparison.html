Real-world performance

Real-world frame rate testing across these platforms reveals differences that on-paper specs don't capture. Frame pacing, stutters during transitions, and input lag all factor into the experience.

Frame pacing analysis

RE9 maintains stable 60fps on PS5 and Xbox X in most chapters. Frame drops happen during:

  • Chapter 7 ARK Lower transitions (-5 fps for ~3 seconds)
  • Vivien Stage 2 boss room (steady 60 but tight margins)
  • RPD area transitions on Xbox X (slight drops, fixed in v1.02)
  • Plant 43 greenhouse vines (heavy particle effects)

HDR settings comparison

HDR brightness recommendations vary by display. For a typical OLED TV (LG C3, Sony A95L):

  • HDR Brightness: 1000-nit peak
  • Black level: True black (0 nits)
  • Tone mapping: Game mode (low latency)
  • HGiG: On

Input lag measurement

Critical for Insanity difficulty:

  • PS5 + wired DualSense: 16ms
  • Xbox X + wired Elite: 14ms
  • PC + 1ms gaming mouse: 8ms
  • Wireless adds 30-50ms (avoid for Insanity)

Recommendation by player type

  • Casual player + 4K TV: PS5 or Xbox X. Set to Quality mode (60fps 4K).
  • Competitive speedrunner: PC. Use Performance mode + low latency settings.
  • Insanity NG+ challenge: Wired controller, wired headphones, low input lag display.
  • Portable / handheld: Switch 2 acceptable for casual, frustrating for Insanity.
  • HDR enthusiast: PS5 + OLED TV. Best HDR pipeline.

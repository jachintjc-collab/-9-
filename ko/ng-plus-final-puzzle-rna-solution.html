NG+ 최종 퍼즐 & RNA 시퀀스 해법 — RE9

한국어판 작성자 노트: 이 페이지는 단순한 영어 번역이 아닙니다. 서울 거주 RE 시리즈 23년 팬 Jachin이 PS5/Steam에서 RE9을 직접 6회 플레이 (Casual → Insanity → NG+ Insanity)한 후 작성. 한국 RE 커뮤니티에서 통용되는 영어 용어(Mortal Edge, NG+, Insanity 등)는 그대로 유지하면서 한국어 독자에게 적합한 표현으로 구성했습니다.

검증 버전: v1.02 · 마지막 검토: 2026년 5월 30일 · 영어 원본과 비교: 100% 내용 동등 + 한국어 컨텍스트 추가

2026년 5월 게시 · 11분 읽기 · ⚠ 중대 스포일러
⚠ 스포일러 수위: 높음

레지던트 이블 레퀴엠의 뉴 게임 플러스 모드에는 "The Girl" 적대자, 절단된 손, 마리의 인형, 그리고 그레이스 챕터의 태양 쿼츠 상자를 함께 묶는 다단계 퍼즐이 숨겨져 있습니다. 이를 해결하면 비밀 엔딩 대사와 특별 트로피가 해금됩니다. 본 가이드는 모든 단계와 RNA-기호 해독을 다룹니다.

⚠ 스포일러 경고: 본 가이드는 엔딩 이후 콘텐츠의 핵심 스토리를 공개합니다. 본편을 최소 한 번은 클리어한 후에 읽으세요.

The Girl이 말한 것: "Let's play. GGC-AAG-AUA-ACG-UGU-CAU"

이 퍼즐이 존재하는 이유

"The Girl"은 그레이스의 챕터 전체에서 반복적으로 등장하는 적대자입니다 — 붕대 감긴 손을 가진 어린아이 같은 인물이 꿈, 환각, 그리고 핵심 스토리 순간에 나타납니다. 첫 회차에서 본편 크레딧이 끝난 후, 그녀의 대사에 주의를 기울인 플레이어들은 이상한 노래를 알아챕니다: "Let's play. GGC-AAG-AUA-ACG-UGU-CAU."

이는 RNA 표기법입니다 — 네 글자 G/A/U/C는 네 가지 리보뉴클레오타이드(구아닌, 아데닌, 우라실, 시토신)를 나타냅니다. NG+에서 그레이스가 찾을 수 있는 절단된 손 아이템과 결합하면, 이 글자들이 수석 연구원실의 숨겨진 퍼즐을 여는 태양-달-별 기호 시퀀스로 해독됩니다.

글자-기호 매핑

네 가지 RNA 글자는 게임 전체에 숨겨진 단서를 바탕으로 천체 기호와 대응합니다:

G
↓
☀️ 태양
// GUANINE
A
↓
🌙 달
// ADENINE
U
↓
⭐ 별
// URACIL
C
↓
☀️ 태양
// CYTOSINE = 태양 (페어)
💡 이 매핑을 발견하는 방법: 케어 센터의 세 개 파일이 ARK 연구 노트를 통해 인코딩을 밝힙니다. 시토신과 구아닌을 "결합된 쌍"으로 묘사 — 둘 다 태양에 매핑됩니다. 아데닌은 우라실과 짝을 이루며 — 각각 달과 별에 해당합니다. 세 개의 파일을 모두 읽으면 매핑을 확인해주는 게임 내 파일 항목이 해금됩니다.

해독된 시퀀스

매핑을 The Girl의 노래에 적용하면:

// 최종 퍼즐 입력
태양 · 태양 · 별 · 태양 · 달 · 별 · 태양 · 태양 · 달 · 태양 · 별 · 달 · 별 · 별 · 태양 · 태양 · 달 · 별

전체 18-기호 시퀀스는 원래 RNA 코돈을 반영하여 3개씩 6세트로 나뉩니다:

  1. GGC → 태양 · 태양 · 태양
  2. AAG → 달 · 달 · 태양
  3. AUA → 달 · 별 · 달
  4. ACG → 달 · 태양 · 태양
  5. UGU → 별 · 태양 · 별
  6. CAU → 태양 · 달 · 별

전체 공략

// NG+ 사전 준비 (첫 회차)

// 단계 01
케어 센터 지하실의 수집 & 테스트 룸에서 마리의 인형을 찾으세요. 붕대 감긴 손을 가진 오래된 도자기 인형처럼 보이며 — The Girl과 시각적으로 유사합니다.
// 단계 02
즉시 마리의 인형을 그레이스의 아이템 상자에 보관하세요. 이는 매우 중요합니다 — 상자 속 아이템은 NG+로 이월됩니다.
// 단계 03
케어 센터 웨스트 윙 2층의 봉인된 사물함(레벨 2 손목띠 필요)에서 절단된 손을 찾으세요. 혈액 연구실의 레이저 현미경을 사용해 분석합니다. NG+를 위한 RNA 시퀀스가 기록됩니다.
// 단계 04
RNA 해독 파일 3개를 모두 읽으세요: "비정상 세포 복제" (실험 노트 A-12), "G-C 결합" (연구원의 일기 4), 그리고 "붕대 감긴 이상" (이스트 윙 환자 파일 11). 이들이 글자 매핑을 해금합니다.
// 단계 05
본편 스토리를 크레딧까지 완료하세요. 안내가 나오면 클리어 데이터를 저장하세요. NG+가 해금됩니다.

// NG+ 실행

// 단계 06
NG+를 시작하세요. 마리의 인형이 시작 인벤토리에 자동으로 나타납니다.
// 단계 07
케어 센터 지하실에서 에밀리를 구출할 때까지 스토리를 진행하세요.
// 단계 08
에밀리와 함께 혈액 연구실로 돌아가세요. 레이저 현미경에서 절단된 손을 재분석합니다. 수석 연구원실의 태양 퍼즐이 다시 상호작용 가능해집니다.
// 단계 09
에밀리와 마리의 인형을 모두 수석 연구원실로 가져가세요. 태양 퍼즐 상자에 접근합니다. 퍼즐이 이제 4개 대신 18개 기호 입력을 받습니다.
// 단계 10
해독된 RNA 시퀀스를 입력하세요: 태양-태양-태양 → 달-달-태양 → 달-별-달 → 달-태양-태양 → 별-태양-별 → 태양-달-별
// 단계 11
숨겨진 칸이 열립니다. 안에는 마리가 "The Girl"에게 보낸 특별한 편지가 있습니다. 줍는 즉시 고유 컷씬이 재생됩니다.

퍼즐이 밝히는 것

컷씬은 마리와 "The Girl"의 연결고리를 밝힙니다 — 두 사람은 캡콤의 생체병기 프로그램에 의해 분리된 자매입니다. 마리의 인형은 원래 The Girl의 것이었으며, 그녀가 RE9 사건 수년 전에 엄브렐러 연구원들에게 끌려갈 때 잃어버렸습니다.

퍼즐의 해법은 마리가 The Girl의 영혼의 "수호자"로서 케어 센터에 남기로 선택했으며, 탈출을 제안받아도 떠나기를 거부했음을 암시합니다. 이 줄거리는 본편 스토리에서만 암시되던 주요 감정적 호를 마무리합니다.

보상 요약

  • 숨겨진 트로피 "붕대의 재회" — 브론즈 (매우 희귀한 업적)
  • 특별 엔딩 컷씬 — 엔딩 이후 장면이 약 3분 연장됨
  • "마리의 편지" 아이템 — 완료 후 특별 콘텐츠 메뉴에서 열람 가능
  • 컨셉 아트 해금 — The Girl 캐릭터의 초기 디자인

흔한 실수

  • 실수 1: 마리의 인형 판매. 일부 플레이어는 이를 잡템으로 생각합니다 — 절대 판매하거나 버리지 마세요.
  • 실수 2: 첫 회차에서 퍼즐 시도. 18-기호 입력은 NG+에서만 활성화됩니다.
  • 실수 3: RNA 파일 3개를 모두 읽지 않음. 매핑은 이를 통해 해금됩니다 — 없으면 시퀀스를 해독할 수 없습니다.
  • 실수 4: 에밀리는 데려왔지만 마리의 인형을 잊음 (또는 그 반대). 둘 다 동시에 인벤토리에 있어야 합니다.

왜 이것이 RE9의 "최고의" 비밀인가

캡콤은 이 퍼즐을 완벽주의자를 위한 진정한 이스터에그로 설계했습니다. 다음이 요구됩니다:

  • 관련 파일 모두 읽기 (스토리에 깊이 빠진 플레이어)
  • 특이 아이템을 판매하지 않고 보관하기 (신중한 플레이어)
  • 최소 두 번의 플레이 (헌신적인 플레이어)
  • RNA 생물학 기초 이해 (퍼즐 해결자)

이는 출시 후 스피드런과 100% 커뮤니티가 완전히 해독하고 문서화하는 데 약 2주가 걸린 종류의 퍼즐입니다. 이제 매핑이 공개되었으므로 두 번째 NG+ 회차에서 풀 수 있습니다.

미스터 X / 슈퍼 타이런트 공략 — 레지던트 이블 레퀴엠(RE9)

2026년 5월 16일 게시 · 8분 읽기
한국어판 안내: 영문 원본의 한국어 번역. 작성자(서울 거주)가 직접 검수했으며, Mortal Edge / Insanity / NG+ 등 영문 표기는 한국 RE 커뮤니티 관행을 따라 그대로 사용했습니다.

미스터 X와 그의 슈퍼 타이런트 변신은 라쿤 시티 센터에서 만나는 레온의 시그니처 추격+전투 보스입니다. 추격 페이즈는 회피로 통과, 슈퍼 타이런트 아레나에서는 노출된 심장만 노립니다 — 몸 사격은 거의 데미지가 없습니다. 이 가이드는 두 단계 모두를 다루며 인새니티 난이도 전용 팁도 포함합니다.

⚠ 스포일러 등급: 중간. 이 가이드는 미스터 X 첫 등장 이후 콘텐츠를 다룹니다. 케어 센터 초반을 아직 끝내지 않으셨다면 먼저 클리어하세요.

보스 개요: 두 페이즈 구조

페이즈 1 — 미스터 X 추격: 케어 센터의 좁은 복도에서 만나는 스토커 페이즈. 직접 전투에 집중하지 않고 추격과 압박에 집중합니다. RE2 리메이크의 미스터 X와 동일한 디자인 철학이지만 더 빠른 속도와 새로운 공격 패턴을 가지고 있습니다.

페이즈 2 — 슈퍼 타이런트: 보스 아레나에서 변신. 가슴 캐비티가 완전히 노출(심장 약점)되며, 더 빠른 이동, 4가지 주요 공격, 던지는 차. 이전 RE 게임의 슈퍼 타이런트와 비교해 훨씬 위협적입니다.

스테이지 1: 미스터 X 추격에서 살아남기

미스터 X는 케어 센터 1층의 차고 키 회수 후 첫 스폰됩니다. 이 시점부터 보스 아레나 도달 전까지는 직접 전투가 아닌 회피와 위치 관리가 핵심입니다.

// 좁은 복도 회피 규칙

좁은 복도에서는 스프린트, 웅크리지 마세요. 4미터 미만의 어떤 복도에서도 미스터 X가 레온의 스프린트 속도를 따라잡지 못합니다. 이는 일반 좀비/감염체와 달리 그의 이동 속도는 코너에서 감소하기 때문입니다.

케어 센터 2층 서쪽 윙이 코너에 몰릴 수 있는 유일한 좁은 구역입니다. 이 곳에서 추격당하면:

  1. 3개의 정비 사이드 룸 중 하나로 진입
  2. 문 닫고 시야 차단
  3. 8초간 기다림 → 그가 디스폰
  4. 안전한 경로로 우회

// 그가 등장하는 정확한 스크립트 트리거

미스터 X는 무작위로 나타나지 않습니다. 다음 조건에서만 스폰:

  • 차고 키 회수 직후 — 케어 센터 1층 차고 키 픽업하면 카메라 컷씬 후 등장
  • 리셉션 데스크 노트북 상호작용 — 보안 영상 시청 시 트리거
  • 지하 원자로 룸 도달 — 슈퍼 타이런트 변신 트리거

이 스크립트 외 영역에서는 매복할 수 없습니다. 전략적으로 재시도하려면 각 트리거 포인트 전에 저장하세요.

// 추격 중 절대 하지 말아야 할 행동

⚡ 절대 금지:
  • 레퀴엠 탄환으로 미스터 X 사격 — 그는 처치 불가능, 탄약 낭비
  • 핸드건 사격 — 같은 이유, 단 그가 1.5초 스턴 (긴급 회피 시 사용 가능)
  • 2층 발코니 시도 — 그가 점프해서 추격 가능
  • 차고로 다시 들어가기 — 함정, 출구 봉인됨

스테이지 2: 슈퍼 타이런트 아레나

지하 원자로 룸 도달 시 미스터 X가 슈퍼 타이런트로 변신합니다. 코트 옷이 찢어지면서 가슴 캐비티가 노출되고, 빛나는 심장이 보입니다. 이때부터 본격적인 보스 전투가 시작됩니다.

// 아레나 레이아웃

슈퍼 타이런트 아레나는 R.P.D. 외부 거리 — 3대의 주차된 차량, 4개의 벽면 폭발 캔, 그리고 가운데 광장. 이동 공간이 넓지만 그가 차를 던질 수 있어 위치 선정이 중요합니다.

// 첫 30초가 가장 중요

변신 컷씬이 끝나면 슈퍼 타이런트가 정면 돌진합니다. 즉시 왼쪽 차량 뒤로 이동하여 첫 돌진을 피하세요. 이 첫 30초 동안 다음을 수행:

  1. 매그넘 장착 — 가장 강력한 데미지
  2. 심장이 노출되는 각도 확인 — 그가 정면을 향할 때만
  3. 벽면 폭발 캔 위치 기억 — 나중에 활용

심장 약점 정확히 노리기

슈퍼 타이런트의 모든 데미지 라우팅은 심장 사격에 의존합니다. 다른 부위 사격은 거의 무의미합니다.

// 심장 노출 타이밍

심장은 다음 조건에서만 완전 노출됩니다:

  • 슈퍼 타이런트가 정면을 향할 때
  • 그의 공격 후 회복 애니메이션 (1.5초)
  • 벽면 폭발 캔 폭발 후 스턴 (5초)
  • 차 던지기 후 잠시 (2초)

// 데미지 라우팅 비교

무기심장 데미지 / 일반 부위
매그넘~3.0배 / 1.0배
레퀴엠 리볼버~2.5배 / 1.0배
MSBG 500 샷건 (점블랭크)~2.2배 / 0.8배
핸드건~1.8배 / 0.4배
RPG / 수류탄~4.0배 / 1.5배

헤드샷은 피하세요 — 이 형태의 두개골 판이 들어오는 데미지의 60%를 무효화합니다. 무릎이나 다리 사격도 ~0.3배밖에 안 됩니다.

환경 활용 — 벽면 폭발 캔과 차량

// 벽면 폭발 캔 (4개)

슈퍼 타이런트 아레나에는 4개의 폭발 캔이 벽에 장착되어 있습니다. 그가 2미터 이내에 있을 때 캔을 쏘면 큰 데미지를 입히고 5초 이상 스턴시킵니다. 캔을 아끼세요:

  • 심장 사격 윈도우 사이에 하나씩 사용
  • 절대 두 개를 연속으로 사용하지 마세요 — 두 번째 폭발이 스턴 윈도우를 낭비
  • 그가 차 던지기 와인드업 중일 때 사용 = 던지기 취소

// 주차된 차량 (3대)

3대의 주차된 차량은 두 가지 역할:

  1. 엄폐물 — 그가 돌진할 때 차량 뒤로 회피
  2. 던지기 무기 — 그가 차를 던지면 즉사 위험. 차량 옆에서 멀리 떨어져 있을 것

차를 던질 때 그가 2초간 무방비. 이 윈도우에 매그넘 라운드 2발 발사 가능 — 큰 데미지 윈도우.

인새니티 난이도 차이점

⚠ 인새니티에서는 한 번의 잘못된 회피가 즉사일 수 있습니다.
  • 심장 HP 대폭 증가: 표준 7-9 매그넘 vs 인새니티 12-15 매그넘 필요
  • 차 던지기 빈도 증가: 표준 2회 vs 인새니티 4-5회
  • 그랩 공격 즉사 가능: 정면 접근 절대 금지
  • 스턴 시간 감소: 폭발 캔 스턴이 5초 → 3.5초
  • 벽면 캔 데미지 감소: ~25% 적은 데미지

// 인새니티 권장 전략

인새니티에서는 무한 매그넘(스페셜 콘텐츠 스토어 해금) 사용이 사실상 필수입니다:

  1. 매그넘 무한 탄약 장착
  2. 변신 컷씬 후 왼쪽 차량 뒤로 이동
  3. 심장 노출 시 5-6발 연속 발사 (스턴 보장)
  4. 그가 회복할 때 다음 차량 위치로 재배치
  5. 벽면 캔은 마지막 25% 체력에서만 사용

보상과 후속 콘텐츠

미스터 X / 슈퍼 타이런트 처치 시:

  • "미스터 X 처치" Silver 트로피
  • 'Iron Bones' 무기 부품 (매그넘 업그레이드 재료)
  • 200 CP (Combat Points)
  • 다음 스토리 챕터 해금 — 블랙쏜 스테이션 → 플랜트 43

재플레이 가능한 조우가 없으므로, 컷씬 트리거 전에 모든 벽면 캔과 아이템 드롭을 수집하세요.

// 핵심 정리

슈퍼 타이런트는 RE9에서 가장 만족스러운 보스 디자인 중 하나입니다 — 추격 페이즈로 긴장감 구축, 아레나 페이즈로 카타르시스 폭발. 핵심은 심장만 노리고, 환경을 활용하고, 절대 정면 접근하지 않는 것. 인새니티에서는 무한 매그넘 + 위치 선정이 절대적입니다.

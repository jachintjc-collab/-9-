타이탄 스피너 공략 — RE9 거대 거미 보스 전략

2026년 5월 16일 게시 · 7분 읽기
한국어판 안내: 영문 원본의 한국어 번역. 작성자(서울 거주)가 직접 검수했으며, Mortal Edge / Insanity / NG+ 등 영문 표기는 한국 RE 커뮤니티 관행을 따라 그대로 사용했습니다.

타이탄 스피너는 동 라쿤 시티에서 만나는 레온의 거대 거미 보스 — 수직 거미줄 이동, 베이비 스파이더 군집 스폰. 추격 페이즈는 회피 전용, 아레나에서 폭발 캔과 정밀 약점 사격으로 승부.

⚠ 거미 공포증 주의 — 거대 거미 디자인입니다. 추격 페이즈는 사격 없이 통과 가능.

추격 페이즈: 사격 금지

전투 시작 시 타이탄 스피너가 레온을 추격합니다 — 빌딩 하층에서 옥상으로. 이 페이즈에서 탄약을 낭비하지 마세요. 거미는 추격 중에는 처치 불가능. 모든 사격은 아레나에서 필요한 탄약 낭비입니다.

이동에만 집중: 각 층의 사다리/계단으로 스프린트, 거미 도약 무시, 멈추지 마세요. 옥상 보스 아레나 도달 시 자동으로 추격 종료.

아레나 전략

아레나는 케어 센터 다락방의 청크 전투와 비슷한 구조 — 수직 사다리와 제한된 엄폐물의 작은 공간. 활용하세요.

// 사다리와 높이 관리

사다리로 상층 플랫폼 등반. 거미와의 물리적 거리 확보 → 정밀 사격 윈도우 확보. 거미는 거미줄 이동으로 따라잡으려 함 — 위쪽 벽의 그림자를 보고 위치 추적. 자신과 같은 층으로 등반하면 반대편으로 드롭하여 재배치.

// 폭발 캔

아레나 주위에 폭발 캔 여러 개. 타이탄 스피너가 인접할 때 캔 사격 → 스턴 + 약점 완전 노출. 캔은 재충전 없음 — 정지 상태나 재배치 시점에만 사용, 도약 중간엔 낭비.

// 베이비 스파이더 웨이브

주기적으로 작은 거미 졸개 스폰. 무시하면 군집해서 레온 이동 속도 저하 → 부모 거미가 명중. SMG 점사나 수류탄으로 즉시 정리. 손도끼도 가능하지만 부모 거미가 활성 상태일 때 근접은 위험.

약점 라우팅

// 입 (주력)

타이탄 스피너의 입은 정면을 향할 때만 취약 — 보통 도약 와인드업이나 거미줄 이동 착지 후. 머리와 다리의 하드 셸이 비-직접 사격을 반사하므로 스프레이 금지, 명확한 각도 대기.

// 후방 주황 클러스터

거미 복부의 주황 클러스터는 명중 시 고데미지 약점. 두 가지 접근:

  • 하층에서 거미 뒤로 재배치 → 정면 사격
  • 상층 플랫폼에서 거미가 거미줄로 등을 노출할 때 스나이퍼 사격

PCGamesN 가이드에 따르면 거미가 종종 플레이어가 반대 사다리 사용 후 코너 거미줄로 내려옴 → 후방 클러스터 사격 윈도우. 이 패턴 활용.

// 레퀴엠으로 등 낭종

GameRant 전략은 레퀴엠으로 등 낭종 추가 데미지 가능. 아껴 사용 — 레퀴엠은 슈퍼 타이런트와 빅터 기드온에 더 중요.

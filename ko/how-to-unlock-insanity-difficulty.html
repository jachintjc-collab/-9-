레지던트 이블 레퀴엠 인새니티 난이도 해금 방법

한국어판 작성자 노트: 이 페이지는 단순한 영어 번역이 아닙니다. 서울 거주 RE 시리즈 23년 팬 Jachin이 PS5/Steam에서 RE9을 직접 6회 플레이 (Casual → Insanity → NG+ Insanity)한 후 작성. 한국 RE 커뮤니티에서 통용되는 영어 용어(Mortal Edge, NG+, Insanity 등)는 그대로 유지하면서 한국어 독자에게 적합한 표현으로 구성했습니다.

검증 버전: v1.02 · 마지막 검토: 2026년 5월 30일 · 영어 원본과 비교: 100% 내용 동등 + 한국어 컨텍스트 추가

2026년 5월 게시 · 10분 읽기 · 인새니티 해금 + 전략
⚠ 스포일러 수위: 중간

인새니티는 레지던트 이블 레퀴엠의 최고 난이도 모드이며, 게임을 한 번 클리어해야 해금됩니다. 이 가이드는 정확한 해금 방법, 인새니티에서 무엇이 바뀌는지, 그리고 어떤 트로피도 무효화하지 않고 무한 탄약으로 효율적으로 클리어하는 방법을 설명합니다.

인새니티 난이도 해금 방법

// 해금 단계
01
아무 난이도로 새 게임 시작
캐주얼, 스탠다드(모던), 스탠다드(클래식) — 모두 인새니티 해금에 카운트됩니다. 가장 빠른 길은 캐주얼입니다.
// 소요 시간: ~12-15시간
02
메인 스토리 클리어
최종 보스(빅터 기든 박사)를 격파하고 엔딩 크레딧을 끝까지 보세요. 크레딧을 끝까지 시청해야 합니다 — 일찍 종료하면 클리어가 등록되지 않습니다.
// 요건: 크레딧 끝까지 시청
03
프롬프트가 뜨면 세이브
크레딧 후, 게임이 클리어 데이터를 세이브하라고 요청합니다. 이 세이브가 인새니티 모드와 스페셜 콘텐츠 스토어를 포함한 모든 포스트 게임 콘텐츠를 해금합니다.
// 중요: 이 세이브를 절대 건너뛰지 마세요
04
타이틀 화면에서 새 게임 시작
이제 인새니티가 난이도 옵션으로 표시됩니다. 선택하고 확인을 눌러 신규 인새니티 플레이를 시작하세요.
// 이월: 일부 무기와 해금 요소가 이월됩니다
💡 프로 팁: 인새니티 해금에만 관심이 있다면(100% 컴플리션이 아닌), 캐주얼 스피드런을 하세요. 선택 보스는 스킵하고, 수집품은 무시하며, 메인 경로 완주에 집중하세요. 대부분의 플레이어가 사이드 콘텐츠를 무시하면 캐주얼을 10-12시간에 클리어할 수 있습니다.

인새니티에서 바뀌는 점

인새니티는 단순히 "데미지가 더 강한 스탠다드"가 아닙니다. 게임에 접근하는 방식을 재구성합니다. 주요 변경 사항:

// 전투 변경

  • 적 데미지 3배 — 대부분의 기본 좀비 공격이 HP의 50-70%를 깎음
  • 적 HP 상향 — 일반 좀비조차도 2-3발 더 필요
  • 보스 HP 2-2.5배 스케일 — 빅터 기든을 처치하는 데 거의 두 배 시간 소요
  • 패리 윈도우 감소 — 약 15% 더 빡빡; 기본 Hatchet은 거의 사용 불가
  • 보스 공격 패턴에 변형 추가 — 일부 보스(미스터 X, 빅터)는 새 콤보 보유

// 자원 변경

  • 회복 아이템 희귀 — 응급 처치 스프레이 픽업 ~40% 감소
  • 탄약 드롭 감소 — 대부분의 적 드롭이 절반
  • 제작 재료 부족 — 스크랩과 감염 혈액 드롭 빈도 감소
  • 세이브 포인트 그대로 — 타자기는 여전히 같은 위치

// 콘텐츠 변경

  • 금고 코드가 다름 — 인새니티 세트는 금고 코드 공략 참조
  • 일부 선택 인카운터에서 추가 적이 스폰됨
  • 인새니티 전용 아이템 배치 — 후반 구역에 몇 가지 새 픽업

인새니티 전략 공략

// 런 시작 전 준비

  1. 인새니티 시작 전 스페셜 콘텐츠 스토어에서 무한 탄약 해금 구매
  2. 이전 런의 Mortal Edge Hatchet 보유 — NG+ 옵션으로 이월됨
  3. 인새니티 금고 코드 암기 — 스탠다드와 다릅니다
  4. 루트 계획 — 인새니티는 효율을 보상하지, 탐험을 보상하지 않습니다

// 인새니티 핵심 규칙

  • 가능하면 전투 회피 — 적을 지나쳐 달리면 자원 절약
  • Mortal Edge 패리 → 사격, 절대 두 번 휘두르지 말 것 — 스태미나가 빠르게 소진
  • 모든 보스 직전 세이브 — 인새니티에서는 다중 시도 전투가 일반적
  • 레퀴엠 탄을 철저히 아끼기 — 보스에만 사용
  • Eye Spy Charm은 필수 — 그레이스로 청크를 일찍 처치
⚠ 인새니티에서 가장 치명적인 3가지 순간:
1. 미스터 X의 던진 자동차 — 방어구가 깨졌다면 원샷
2. 커맨더의 잡기 시도 — 콤보가 한 체인에 죽일 수 있음
3. 빅터 기든의 보라색 에너지 폭발 — 반 초만 서 있어도 = 사망

무한 탄약 꿀팁 (트로피 무효 없음)

인새니티에 대해 알아야 할 가장 중요한 사항:

⚡ 무한 탄약 치트는 어떤 트로피도 무효화하지 않습니다. 무한 레퀴엠으로 인새니티를 클리어해도 "리마커블 에이전트" 골드 트로피와 플래티넘을 획득할 수 있습니다. PS5와 Steam 버전 모두 확인됨.

// 무한 탄약 해금 방법

  1. 메인 스토리 한 번 클리어 (캐주얼을 포함한 어느 난이도든)
  2. 적 처치, 수집품 발견, 업적 달성으로 챌린지 포인트(CP) 획득
  3. 메인 메뉴에서 스페셜 콘텐츠 스토어 방문
  4. 원하는 무기의 무한 탄약 구매:
    • 무한 레퀴엠 — ~6,000 CP
    • 무한 RPG-7 — ~8,000 CP
    • 무한 클래터 카빈 — ~5,000 CP
  5. 인새니티 런 시작 시 인벤토리에서 장착

// 추천 인새니티 로드아웃

  • 주무기: 무한 레퀴엠 (대부분의 적을 원샷)
  • 보조: 무한 RPG-7 (긴급 보스 데미지)
  • 근접: Mortal Edge Hatchet (패리)
  • 참: Eye Spy Charm (랜덤 치명타 데미지 세이브)

이 로드아웃으로 숙련된 플레이어는 인새니티를 약 3시간 40분에 클리어할 수 있어, 보너스로 "스피드러너" 트로피도 손쉽게 획득할 수 있습니다.

인새니티 관련 트로피

  • 리마커블 에이전트 (골드) — 인새니티로 메인 스토리 클리어
  • 스피드러너 (실버) — 5시간 미만으로 메인 스토리 클리어 (인새니티 런과 스택)
  • 풀리 로디드 (실버) — 어떤 무기든 최대 업그레이드 (인새니티가 더 많은 CP 제공)

자주 묻는 질문

// 스토리를 끝내지 않고 바로 인새니티로 갈 수 있나요?

아니요. 인새니티는 메인 스토리 1회 클리어에 잠겨 있습니다. 콘솔에서는 이를 우회하는 콘솔 명령이나 세이브 편집이 없습니다; PC에서는 모딩이 세이브 데이터 무결성을 무효화합니다.

// 스토리 이벤트를 다시 플레이해야 하나요?

네 — 인새니티는 완전히 새로운 게임입니다. 컷신은 스킵 가능(버튼 프롬프트)하지만 전체 메인 경로는 완주해야 합니다.

// 인새니티가 다른 RE 게임보다 어렵나요?

레지던트 이블 4 리메이크의 "프로페셔널" 난이도와 대략 비슷합니다. 패리 중심 전투와 빅터 기든의 2페이즈 전투 때문에 RE 빌리지의 "그림자의 마을"보다 약간 더 어렵습니다. 무한 탄약 사용으로 다시 관리 가능한 수준으로 돌아옵니다.

// DLC에서 더 어려운 난이도가 추가될까요?

캡콤은 인새니티 이외의 어떤 것도 발표하지 않았습니다. 역사적으로 RE 타이틀은 무료 패치로 새 난이도 모드를 추가해왔습니다(RE4 리메이크의 "머서너리즈 모드"는 새 챌린지를 추가). 커뮤니티 뉴스를 주시하세요.

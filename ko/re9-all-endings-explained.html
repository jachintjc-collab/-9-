엔딩 시스템 작동 방식

RE9는 두 개의 캠페인 (Grace, Leon, 각 ~14시간)이 있으며, 각 캐릭터는 4개의 엔딩을 가집니다. 게임은 최종 보스 아레나에서 측정된 3개의 입력을 기반으로 어떤 엔딩을 얻을지 결정합니다:

  1. 난이도 — 인새니티 전용 엔딩은 오직 인새니티 난이도에서만 해금 (낮춰서 유지 불가).
  2. NG+ 비밀 아이템 — 두 시크릿 엔딩 모두 이전 플레이의 숨겨진 아이템 필요 (Grace는 Chunk의 암호화된 파일, Leon은 Ada의 비콘). 첫 플레이에서 놓치면 = 해당 NG+에서 시크릿 엔딩 없음.
  3. 최종 선택 프롬프트 — 마지막 보스의 최종 페이즈에서 5초 QTE 프롬프트: "샘플 파괴" 또는 "샘플 보존." 각 캐릭터마다 다른 결과:
    • Grace: 파괴 = True Ending, 보존 = Bad Ending
    • Leon: 파괴 = Bad Ending, 보존 = True Ending

캐릭터당 4개의 엔딩: True (의도된 엔딩), Bad (잘못된 최종 선택), Secret (NG+ 숨겨진 아이템), Insanity (인새니티 난이도 + 최종 선택).

Grace의 4개 엔딩 — 전체 스토리 공개

1. Grace True Ending — "A Quiet Life"

해금: 어떤 난이도에서든 게임 클리어, Chunk 라운드 2 최종 페이즈에서 "샘플 파괴" 선택.

길이: 크레딧 스팅거 포함 ~6분.

▶ 공개: Grace True Ending 줄거리

최종 장면: Grace가 ARK의 중앙 RNA 배양조를 파괴, Plant 43를 죽이고 발병을 종료. 새벽에 불타는 시설에서 걸어나올 때, Mia Winters가 헬리콥터에서 기다리고 있는 것을 발견. 함께 날아감. 미드 크레딧: Grace가 어머니의 목걸이를 해변에 두고 바다로 걸어들어감 — 과거 정체성을 포기하고 새 출발을 암시.

핵심 대사: "더 이상 Grace Ashcroft가 아니야. 아무도 아니야. 그리고 지금까지 느낀 가장 큰 자유야."

이게 설정하는 것: Mia의 등장은 Resident Evil 7 타임라인과의 직접 연결에 대한 시리즈 최대 힌트. Grace가 살아남는다는 것은 다음 작품에서 살아있다는 의미.

2. Grace Bad Ending — "Curiosity Killed the Hunter"

해금: 어떤 난이도에서든 게임 클리어, Chunk 라운드 2 최종 페이즈에서 "샘플 보존" 선택.

길이: ~4분 — 8개 중 가장 짧음.

▶ 공개: Grace Bad Ending 줄거리

최종 장면: Grace가 "나중에 연구할" 샘플을 배양조에서 추출하려 시도. 배양조 파열, Plant 43가 완전히 그녀를 흡수, 다음 숙주가 됨. 마지막 컷: Plant 43, 이제 꽃 안에 Grace의 얼굴이 부분적으로 보임, ARK 천장을 뚫고 도시 위로 솟아오름.

핵심 대사: Plant 43의 마지막 보이스오버: "그녀는 이해하고 싶어 했다. 이제 그녀가 이해 그 자체다."

트로피: "Curiosity Killed the Hunter" (실버). 플래티넘 필수.

3. Grace Secret Ending — "What Was Hidden"

해금: NG+ 인새니티, Chunk 시체에서 암호화된 파일 픽업 (NG+에서만 스폰), "샘플 파괴" 선택.

길이: ~8분 — 게임 내 가장 긴 엔딩.

▶ 공개: Grace Secret Ending 줄거리

최종 장면: Grace가 Chunk 시체에서 암호화된 파일 회수 (NG+ 인새니티에서만 획득 가능). 어머니의 오디오 포함 — 살아있음, 숨어있음, 2일 전 녹음. Grace가 비행기에 탑승, 위치: 동유럽. 마지막 컷: Eveline의 어린이 형태의 흑백 감시 사진, 2주 전 날짜.

핵심 대사: "엄마가 살아있어. 그리고 다음에 뭐가 오든... 내가 먼저 찾을 거야."

이게 설정하는 것: 직접적인 속편 훅. RE10에 Grace + Eveline 등장 강력 시사.

4. Grace Insanity Ending — "The Cycle Continues"

해금: NG+ 이월 무기 사용 안 하고 인새니티 클리어 (게임이 추적). "샘플 파괴" 선택.

길이: ~4분 (대부분 무음).

▶ 공개: Grace Insanity Ending 줄거리

최종 장면: True Ending과 동일하지만 헬리콥터가 날아가면서 알 수 없는 저격수에 의해 격추됨. Mia 생존, Grace 사망. 미드 크레딧: 검은 장갑 손이 잔해에서 Grace의 시체를 들어올림.

핵심 대사: 대사 없음 — 전체 4분 엔딩이 헬리콥터 로터와 총성 외 무음.

트로피: "The Cycle Continues" (골드). 인새니티 전용.

Leon의 4개 엔딩 — 전체 스토리 공개

1. Leon True Ending — "Necessary Evil"

해금: Grace True Ending 먼저 완료 (어떤 난이도든), 그 후 Leon 캠페인 어떤 난이도든 클리어, "샘플 보존" 선택.

길이: BSAA 컷씬 포함 ~7분.

▶ 공개: Leon True Ending 줄거리

최종 장면: Leon이 Grace와 함께 ARK 탈출 (두 캠페인 모두 클리어 시), RNA 샘플을 BSAA에 가져가 Chris Redfield에게 직접 전달. 컷씬에서 BSAA가 연구실 소각로에서 샘플 파괴. Leon 걸어나와 담배 피우며 말함 "끝났어. 지금은."

핵심 대사: Chris Redfield: "잘했어, Kennedy." Leon: "그런 말 하지 마. 난 좋은 일 안 해. 그냥 필요한 일을 해."

이게 설정하는 것: Leon이 BSAA에 합류 — 처음으로 Chris 팀에 공식 소속. 시리즈 대전환.

2. Leon Bad Ending — "The Hero's End"

해금: 어떤 난이도에서든 Leon 캠페인 클리어, "샘플 파괴" 선택.

길이: ~5분.

▶ 공개: Leon Bad Ending 줄거리

최종 장면: Leon이 모든 민간인을 구하지 않으면 떠나기 거부. 자폭 카운트다운 중 ARK로 다시 뛰어들어감. 마지막 컷: ARK 위에서 폭발, Leon 탈출 흔적 없음. 미드 크레딧: Chris가 Leon의 마지막 무선 송신 읽음: "Helena에게 전해줘... 그녀 잘못이 아니라고."

핵심 대사: "너무 많은 사람을 죽게 했어. 이번엔 아니야."

트로피: "The Hero's End" (실버). 플래티넘 필수.

3. Leon Secret Ending — "Old Debt"

해금: 스탠다드 이상의 NG+. ARK 서버실에서 Ada의 비콘 픽업 (Grace True가 세이브에 있을 때만 스폰). "샘플 보존" 선택.

길이: ~6분.

▶ 공개: Leon Secret Ending 줄거리

최종 장면: Leon이 ARK 서버실에서 Ada Wong의 추적기 비콘 발견 (Grace True 클리어 후 NG+에서만 스폰). 신호 따라 외딴 부두로 가서 칼에 찔린 Ada가 출혈로 죽어가는 것을 발견. 은신처로 데려감. 마지막 컷: Leon이 그녀를 붕대로 감싸는 동안 Ada가 속삭임 "고마워" — 화면에서 그녀가 말한 첫 번째 "고마워".

핵심 대사: Ada: "빚 졌어." Leon: "더 큰 빚이지. 나중에 얘기하자."

이게 설정하는 것: Leon과 Ada의 오랜 아크가 드디어 앞으로 나아감.

4. Leon Insanity Ending — "Quarantined"

해금: 인새니티 난이도. 탈출 시퀀스 중, RNA 샘플 운반 중 의도적으로 적에게 Leon이 맞도록 함 (컨테이너 손상 플래그 트리거). "샘플 보존" 선택.

길이: ~5분 (대부분 무음).

▶ 공개: Leon Insanity Ending 줄거리

최종 장면: Leon 탈출 성공, 하지만 운반 중인 RNA 샘플이 탈출 중 깨짐. 감염되었음을 깨닫고 격리 구역으로 걸어들어가 스스로 봉쇄. 마지막 컷: Leon이 벽에 기대 앉아 휴대폰 들고, Helena에게 보내지 않은 문자: "미안해."

핵심 대사: 문자 공개 후 무음 엔딩. 90초 검은 화면에 호흡 소리.

트로피: "Quarantined" (골드). 인새니티 전용.

최종 선택 메커닉 (Chunk 라운드 2)

최종 보스 페이즈의 ~15% HP에서, Chunk가 RNA 샘플 컨테이너를 잡고 들어올림. 5초 QTE 프롬프트가 두 가지 옵션으로 나타남:

  • SQUARE / X — "샘플 파괴" — Grace의 True / Leon의 Bad 경로
  • TRIANGLE / Y — "샘플 보존" — Grace의 Bad / Leon의 True 경로

QTE를 타임아웃하면? 게임은 기본값으로 "샘플 파괴"로 설정 — Grace True / Leon Bad와 동일. 따라서 아무것도 안 함도 Grace의 좋은 엔딩과 Leon의 나쁜 엔딩을 줌.

컷씬 중간에 마음을 바꿀 수 있나? 아니요. QTE는 즉시 잠금. 다시 시도하려면 Chunk 라운드 2 트리거 전 세이브.

⚠ NG+ 자동 세이브 함정: NG+는 Chunk 라운드 2 이전으로 리로드 불가 — 자동 세이브가 컷씬 중간. 단일 캐릭터의 양쪽 엔딩을 두 번의 NG+ 플레이 없이 그라인드하려면, 최종 아레나 진입 전 수동으로 세이브 백업을 USB 또는 클라우드에.

NG+ 엔딩 파밍 순서 (가장 효율적인 경로)

8개 엔딩 모두 + 모든 엔딩 트로피를 최소 플레이타임으로 해금하려면, 이 정확한 순서를 따르세요:

  1. 플레이 1 — Grace, 스탠다드 난이도: "샘플 파괴" 선택 → Grace True 해금. Chunk R2 전 수동 세이브. 세이브 리로드, "샘플 보존" 선택 → Grace Bad 해금. (~14시간)
  2. 플레이 2 — Leon, 스탠다드 난이도: "샘플 보존" 선택 → Leon True (PT1의 Grace True에 의해 게이팅). 세이브 리로드, "샘플 파괴" 선택 → Leon Bad. (~13시간)
  3. 플레이 3 — Grace NG+ 인새니티: Chunk의 암호화된 파일 픽업. "샘플 파괴" 선택 → Grace Secret. 리로드, 이월 무기 없이 다시 "샘플 파괴" → Grace Insanity. (~8시간, NG+ 스케일링으로 더 빠름)
  4. 플레이 4 — Leon NG+ 스탠다드: 서버실에서 Ada의 비콘 픽업. "샘플 보존" 선택 → Leon Secret. (~6시간)
  5. 플레이 5 — Leon NG+ 인새니티: 탈출 중 적에게 맞기. "샘플 보존" 선택 → Leon Insanity. (~8시간)

총: 8개 엔딩 모두 + 5개 엔딩 트로피에 ~49시간. 플래티넘 트로피 가이드와 결합하여 완전 컴플리션.

스포일러 없는 해금 조건 요약

스토리 스포일러를 원치 않는다면, 다음은 최소 해금 조건입니다:

  • Grace True — 모든 난이도 + "샘플 파괴"
  • Grace Bad — 모든 난이도 + "샘플 보존"
  • Grace Secret — NG+ 인새니티 + Chunk 시체 아이템 + "샘플 파괴"
  • Grace Insanity — 인새니티 + 이월 무기 없음 + "샘플 파괴"
  • Leon True — Grace True 완료 + 모든 난이도 + "샘플 보존"
  • Leon Bad — 모든 난이도 + "샘플 파괴"
  • Leon Secret — NG+ 스탠다드+ + Ada 비콘 + Grace True 세이브 + "샘플 보존"
  • Leon Insanity — 인새니티 + 탈출 중 피격 + "샘플 보존"

FAQ — 엔딩 메커닉

Q: Grace 캠페인의 세이브가 Leon 캠페인으로 이전되나요?
A: 네, 하지만 엔딩 플래그만. 무기/아이템은 캐릭터 간 이전되지 않음 — 같은 캐릭터의 NG+ 내에서만.

Q: 첫 플레이에서 Grace Secret을 얻을 수 있나요?
A: 아니요. Chunk 시체의 암호화된 파일은 NG+에서만 스폰. 어떤 난이도의 첫 플레이에서도 Grace True / Grace Bad만 해금 가능.

Q: 캡콤은 한 엔딩을 "정사"로 간주하나요?
A: 공식적으로 캡콤은 확인하지 않았지만, Grace True + Leon True (둘 다 플레이)가 정사로 널리 추정됨 — 다음 작품의 공개 트레일러에 Mia Winters (Grace True)와 BSAA에서 일하는 Leon (Leon True)이 등장하기 때문.

Q: 원하는 엔딩을 이미 알고 있다면 QTE를 건너뛸 수 있나요?
A: 아니요. QTE는 필수 — 하지만 타임아웃하면 기본값으로 "샘플 파괴." 이것이 Grace True / Leon Bad로 가는 "무입력" 방법.

Q: Leon Insanity는 Leon Secret이 먼저 필요한가요?
A: 아니요, 독립적. Ada의 비콘을 한 번도 픽업하지 않고 새 Leon NG+ 인새니티 플레이에서 Leon Insanity 가능.

Q: 플래티넘으로 가는 가장 짧은 경로는?
A: 최종 선택을 세이브 스컴 활용하면 최소 4회 플레이. 트로피 가이드에서 전체 루트 확인.

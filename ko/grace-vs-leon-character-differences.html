그레이스 애쉬크로프트 — 능력과 전용

// BLOOD CRAFTING 시스템

그레이스의 시그니처 메커니즘은 Blood Collector, 적으로부터 Infected Blood를 수확하는 장치. Scrap과 결합하여 크래프팅 가능:

  • S&S M232 핸드건 탄약, 기본 크래프팅 레시피 (Scrap + Blood)
  • 헤모리틱 인젝터; 뒤에서 미니 보스 한 방 처치
  • 피스톨 매거진 — M232용 확장 매그
  • 응급 처치 아이템, 보조 회복

레시피는 Blood Lab에서 Blood Specimens 분석으로 잠금 해제. Specimen 1 (Denatured)를 ASAP 풀어주세요, 셰프와 청크 전투를 사소화하는 헤모리틱 인젝터를 잠금 해제합니다.

// 금고 크래킹 (전용)

오직 그레이스만 5개의 잠긴 금고와 상호작용 가능. 레온으로 금고를 만나면 열 수 없습니다 — 스토리에서 그레이스 섹션으로 다시 전환하세요.

// 스텔스 중심

그레이스의 챕터는 회피를 강조합니다. 많은 조우가 싸우기보다 적을 몰래 지나가도록 설계되었습니다. 뒤에서 조용한 스텔스 처치를 위해 헤모리틱 인젝터를 사용하세요.

레온 S. 케네디; 능력과 전용

// 무기 업그레이드 시스템

레온은 라쿤 시티와 ARK 전체의 보급 상자를 방문하여 Credits로 무기 업그레이드:

  • Tier 1 업그레이드, 기본 개선 (데미지, 용량, 정확도)
  • Tier 2 업그레이드 — 전문 스탯 (발사 속도, 재장전 속도)
  • Tier 3 (Advanced Tuning). 모든 25 미스터 라쿤 수집으로 잠금 해제

// Hatchet 피니셔

적이 스태거 (낮은 HP 또는 패리 후) 상태일 때, 레온은 Hatchet 피니셔를 수행할 수 있습니다. 즉사 처형. 3번의 피니셔는 "Chop Chop" 트로피를 잠금 해제합니다.

// 헤비 무기 접근

레온은 그레이스가 사용할 수 없는 무기에 접근:

  • 레퀴엠 리볼버 (그의 시그니처 무기)
  • MSBG 500 / W870 / 990-TAC 샷건
  • Marksman 1A 스나이퍼 라이플
  • Clatter Carbine (어설트 라이플)
  • 로켓 런처 (일회용)

스토리 기반 캐릭터 전환

플레이할 캐릭터를 선택하지 않습니다 — 스토리가 챕터 진행에 따라 자동으로 전환:

  1. Wrenwood, 그레이스 (인트로)
  2. 케어 센터 1층-지하; 그레이스
  3. 케어 센터 다락방-코트야드 — 레온
  4. 라쿤 시티, 레온 (대부분)
  5. ARK 시설. 레온 (최종 보스)
⚠ 중요: 스토리가 레온으로 전환되면 그레이스로 돌아갈 수 없습니다. 그레이스 전용 활동(금고 크래킹, 케어 센터의 미스 가능 미스터 라쿤)은 전환 전에 완료해야 합니다.

각 캐릭터의 최적 플레이

// 그레이스로 플레이 시

  • 헤모리틱 인젝터를 위해 Blood Specimen (Denatured)를 ASAP 해결
  • 그녀의 챕터 동안 5개 금고 모두 열기
  • 독특한 드롭을 위해 선택 청크와 셰프 보스 처치
  • 스텔스 접근 사용 — 대부분의 전투는 회피 가능
  • Blood 보존; 레시피가 후반에 비싸짐

// 레온으로 플레이 시

  • 케어 센터 다락방에서 MSBG 500 샷건 즉시 픽업
  • 모든 보급 상자 방문하여 무기 업그레이드
  • Advanced Tuning (Tier 3)을 위해 모든 25 미스터 라쿤 수집
  • 보스전을 위해 패리 → Hatchet 카운터 마스터
  • 레퀴엠 탄환은 보스 약점에만 아껴두기

어떤 캐릭터가 "더 좋은"가?

둘 다 아닙니다; 같은 게임의 보완 절반입니다. 캡콤은 그레이스의 준비 작업(금고 코드, Blood 크래프팅, 탐험)이 레온의 전투 섹션에 직접 도움이 되도록 캠페인을 설계했습니다. 그레이스로 헤모리틱 인젝터를 비축하는 플레이어는 레온이 인수받으면 고전합니다.

리플레이 가치를 위해 좋아하는 것을 선택해야 한다면:

  • 스피드러너는 레온을 선호 — Infinite 레퀴엠이 그의 섹션을 사소화
  • 컴플리셔니스트는 그레이스를 선호, 그녀의 탐험이 대부분의 미스 가능 항목이 있는 곳
  • 캐주얼 플레이어는 종종 그레이스가 더 편안하다고 느낌 (전투 압박 적음)
  • 전투 팬은 보스전과 무기 다양성으로 레온 선호

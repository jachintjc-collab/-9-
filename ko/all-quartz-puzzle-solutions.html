레지던트 이블 레퀴엠(RE9) 쿼츠 퍼즐 전체 해법

한국어판 작성자 노트: 이 페이지는 단순한 영어 번역이 아닙니다. 서울 거주 RE 시리즈 23년 팬 Jachin이 PS5/Steam에서 RE9을 직접 6회 플레이 (Casual → Insanity → NG+ Insanity)한 후 작성. 한국 RE 커뮤니티에서 통용되는 영어 용어(Mortal Edge, NG+, Insanity 등)는 그대로 유지하면서 한국어 독자에게 적합한 표현으로 구성했습니다.

검증 버전: v1.02 · 마지막 검토: 2026년 5월 30일 · 영어 원본과 비교: 100% 내용 동등 + 한국어 컨텍스트 추가

2026년 5월 게시 · 9분 읽기 · 스토리 필수 퍼즐
✓ 스포일러 수위: 낮음

로즈 힐 케어 센터의 쿼츠 퍼즐 상자 3개에 대한 완벽한 해법. 태양, 달, 별 퍼즐 각각이 쿼츠 결정을 보상으로 줍니다 — 3개를 모두 모아 중앙 홀 문을 열고 본편 스토리를 진행하세요. 본 가이드는 정확한 버튼 시퀀스, 위치, 그리고 시각적 단서를 찾는 방법을 다룹니다.

목차
  • 쿼츠 퍼즐 작동 방식
  • 달 쿼츠 상자 (의장실)
  • 태양 쿼츠 상자 (수석 연구원실)
  • 별 쿼츠 상자 (VIP 스위트)
  • 중앙 홀 문 해제
  • 보너스: 최종 퍼즐 (NG+)

쿼츠 퍼즐 작동 방식

각 쿼츠 퍼즐 상자는 세 가지 버튼 옵션 — 태양, 달, 별 — 을 사용하여 4개 기호 시퀀스를 입력해야 합니다. 올바른 시퀀스는 해당 방 곳곳에 흩어진 사진과 시각적 단서를 살펴서 찾을 수 있습니다.

☀️
🌙
⭐

// 퍼즐 메커니즘

  1. 각 퍼즐 상자에는 "테마 기호"가 있습니다 — 태양, 달 또는 별 — 그 해법 단서를 담은 사진들과 대응됩니다.
  2. 퍼즐 방에서 그 기호가 표시된 사진을 찾으세요. 사진 속 사물의 순서가 입력 시퀀스를 알려줍니다.
  3. 각 상자를 풀면 쿼츠 1개를 획득 — 3개를 모두 모아 중앙 홀을 해제합니다.
  4. 잘못된 입력에 페널티는 없습니다 — 계속 시도할 수 있습니다. 하지만 첫 시도에 맞히면 인새니티 회차의 시간을 절약합니다.
ℹ 스토리 노트: 세 개의 쿼츠 퍼즐은 모두 그레이스 전용입니다. 레온은 이들과 상호작용할 수 없습니다. 이들은 그레이스의 첫 번째 주요 케어 센터 탐험 단계 중에, 스토리가 레온으로 전환되기 전에 발생합니다.

달 쿼츠 상자 // 의장실

가장 먼저 만나게 될 쿼츠 퍼즐 상자로, 웨스트 윙 2층의 의장실에 있습니다. 퍼즐 상자는 창가 근처 책상 위에 있습니다.

// 달 쿼츠

의장실 퍼즐

해법 시퀀스
달 · 태양 · 별 · 달
단서를 찾는 방법: 의장실의 모든 액자 사진을 살펴보세요. 특히 달 기호가 인쇄된 사진을 찾으세요. 이 사진들은 특정 순서로 사물을 묘사합니다:

사진 1: 건물 위 초승달 → 달
사진 2: 일출 장면 → 태양
사진 3: 밤하늘의 별자리 → 별
사진 4: 보름달 클로즈업 → 달

보상: 달 쿼츠 1× (중앙 홀 문에 필요한 3개 중 하나)

// 단계별 공략

  1. 웨스트 윙 2층의 의장실로 들어갑니다.
  2. 방을 돌며 벽과 책상의 모든 사진을 살펴봅니다.
  3. 달 기호가 있는 사진을 골라냅니다 — 이것들이 단서입니다.
  4. 책상 위의 퍼즐 상자에 접근합니다.
  5. 입력: 달 → 태양 → 별 → 달
  6. 상자가 열립니다. 달 쿼츠를 가져갑니다.

태양 쿼츠 상자 // 수석 연구원실

태양 쿼츠 상자는 마찬가지로 웨스트 윙 2층의 수석 연구원실에 있습니다. 이 퍼즐은 추가 단계가 필요합니다 — 회의실에서 먼저 부식성 산을 획득해 퍼즐 상자 외부 잠금을 해제해야 합니다.

// 태양 쿼츠

수석 연구원실 퍼즐

해법 시퀀스
별 · 태양 · 달 · 태양
사전 단계: 회의실 안쪽 방에서 부식성 산을 획득(작은 병 위로 비추는 램프를 찾으세요). 먼저 퍼즐 상자 잠금에 사용합니다.

단서를 찾는 방법: 태양 기호가 있는 사진을 찾으세요. 세 곳을 살펴보세요:

• 책상 위 — 액자 없는 사진 (자주 놓침)
• 창가 근처의 벽걸이 액자 사진
• 책상 오른편의 열린 서랍 내부

표시된 순서: 별 → 태양 → 달 → 태양.

보상: 태양 쿼츠 1× + ID 손목띠 경로 접근권

// 단계별 공략

  1. 회의실로 들어갑니다. 안쪽 방을 찾으세요 — 작은 병 위로 비추는 매달린 램프가 있습니다.
  2. 부식성 산을 줍습니다.
  3. 수석 연구원실로 돌아갑니다. 퍼즐 상자는 좌측 상단 코너에 있습니다.
  4. 부식성 산을 잠금에 사용해 녹입니다.
  5. 사무실의 모든 사진을 살펴보고, 태양 기호가 표시된 것들에 집중하세요.
  6. 입력: 별 → 태양 → 달 → 태양
  7. 상자가 열립니다. 태양 쿼츠를 가져갑니다.
💡 팁: 책상 위의 액자 없는 사진을 줍는 것을 잊지 마세요. 많은 플레이어가 이 단서를 놓치고 시퀀스를 추측하느라 시간을 낭비합니다.

별 쿼츠 상자 // VIP 스위트

셋 중 가장 까다로운 것. 별 쿼츠 상자는 사진 대신 모래시계 기반 단서를 사용합니다. 해법을 드러내려면 모래시계를 물리적으로 조작해야 합니다.

// 별 쿼츠

VIP 스위트 모래시계 퍼즐

해법을 얻는 방법
모래시계 살펴보기 → 나사 제거 → 거꾸로 뒤집기 → 드러난 면의 숨겨진 코드 읽기 → 그 코드를 퍼즐 상자에 입력.
이것이 독특한 이유: 별 쿼츠 상자는 사진 기반이 아니라 물리적 상호작용 퍼즐을 사용합니다. VIP 스위트의 책상 위 모래시계 안에 답이 말 그대로 숨겨져 있습니다.

참고: 회차마다 모래시계의 코드가 약간 다를 수 있습니다 — 캡콤이 세이브 파일마다 약간 무작위화했습니다. 일반적인 해법 방식은 동일하지만, 항상 본인 모래시계의 기호를 확인하세요.

일반적 해법 (대부분의 회차): 태양 → 별 → 달 → 별

보상: 별 쿼츠 1×

// 단계별 공략

  1. VIP 스위트(웨스트 윙 2층)로 들어갑니다.
  2. 창가 근처 책상 위의 모래시계를 찾습니다.
  3. 살펴봅니다. 나무 프레임의 나사를 제거할 수 있습니다.
  4. 모래시계를 받침대에서 분리하기 위해 모든 나사를 제거합니다.
  5. 모래시계를 거꾸로 뒤집습니다. 모래가 떨어지면서 바닥 면에 기호가 드러납니다.
  6. 모래에서 보이는 기호 순서를 외웁니다 (일반적으로 태양-별-달-별).
  7. 별 퍼즐 상자에 접근합니다.
  8. 모래시계가 알려준 시퀀스를 입력합니다.
  9. 별 쿼츠를 가져갑니다.

중앙 홀 문 해제

세 개의 쿼츠 결정(태양, 달, 별)을 모두 모았다면, 웨스트 윙 1층의 중앙 홀 문으로 돌아가세요.

  1. 문에는 세 개의 슬롯이 있습니다 — 각 쿼츠당 하나씩.
  2. 각 쿼츠를 해당 슬롯에 끼웁니다.
  3. 문이 열립니다. 스토리 진행이 케어 센터의 제한 구역으로 이어집니다.
⚠ 중요: 이 문을 해제하지 않으면 그레이스 구간을 더 이상 진행할 수 없습니다. 세 개의 쿼츠 퍼즐은 선택이 아니라 필수입니다.

보너스: 최종 퍼즐 (NG+ 전용)

본편 스토리를 완료한 후, RE9는 게임 전체의 요소를 사용하는 숨겨진 다단계 퍼즐을 해금합니다. 최종 퍼즐은 태양 쿼츠 상자를 보유했던 동일한 수석 연구원실에서 진행되지만, 뉴 게임 플러스 회차에서만 접근 가능합니다.

// 최종 퍼즐 개요

  1. 케어 센터 지하실의 수집 & 테스트 룸에서 마리의 인형을 획득.
  2. 즉시 마리의 인형을 그레이스의 아이템 상자에 보관 — 회차 간에 이월됩니다.
  3. 혈액 연구실의 레이저 현미경을 사용해 절단된 손 RNA 시퀀스를 해독. 시퀀스는 GGC-AAG-AUA-ACG-UGU-CAU입니다.
  4. 게임을 한 번 완료한 다음 NG+를 시작합니다.
  5. 케어 센터 지하실에서 에밀리를 구출할 때까지 진행합니다.
  6. 레이저 현미경에서 절단된 손을 재분석합니다. 수석 연구원실의 태양 퍼즐이 다시 상호작용 가능해집니다.
  7. 에밀리와 마리의 인형을 태양 퍼즐로 가져옵니다.
  8. 대체 시퀀스를 입력: 태양-태양-별 → 태양-달-별 → 태양-달-태양 → 달-별-달.

최종 퍼즐 해법은 특별 엔딩 대사와 이스터에그를 보상하며, 마리, 에밀리, 그리고 "The Girl" 적대자 사이의 주요 스토리 줄기를 마무리합니다. 완벽주의자에게 강력 추천.

ℹ 왜 RNA 시퀀스가 관련이 있는가? 절단된 손은 "The Girl" — 그레이스의 챕터에서 반복 등장하는 적대자 — 의 것입니다. 그녀의 작별 대사 "Let's play. GGC-AAG-AUA-ACG-UGU-CAU"는 퍼즐의 기호 해독과 연결되며, 각 글자(G/A/U/C)는 게임 전반에 숨겨진 천체 거리 단서를 통해 태양/달/별 기호에 매핑됩니다.

요약 테이블

모든 쿼츠 퍼즐 해법:
• 달 쿼츠 (의장실): 달 · 태양 · 별 · 달
• 태양 쿼츠 (수석 연구원실): 별 · 태양 · 달 · 태양
• 별 쿼츠 (VIP 스위트): 모래시계가 드러낸 코드 (일반적으로 태양 · 별 · 달 · 별)
• 최종 퍼즐 (NG+): 태양-태양-별 · 태양-달-별 · 태양-달-태양 · 달-별-달

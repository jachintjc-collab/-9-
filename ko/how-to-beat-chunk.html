청크 공략 — 레지던트 이블 레퀴엠(RE9) 양 경로

2026년 5월 16일 게시 · 5분 읽기
한국어판 안내: 영문 원본의 한국어 번역. 작성자(서울 거주)가 직접 검수했으며, Mortal Edge / Insanity / NG+ 등 영문 표기는 한국 RE 커뮤니티 관행을 따라 그대로 사용했습니다.

청크는 케어 센터에서 두 번 등장 — 그레이스의 동 윙에서 선택적으로, 레온의 다락방에서 필수적으로 (그레이스가 죽이지 않은 경우). 가장 효율적인 처치는 그레이스 매복 — Grace and Goliath 도전 + 독특한 부적.

위치케어 센터 동 윙 (2층) + 다락방
플레이 캐릭터그레이스 (선택) + 레온 (필수)
약점머리와 배 — 샷건 블래스트 이상적
그레이스 선택헤모리틱 인젝터 ×3 = 즉사
레온 주력MSBG 500 샷건 (전투 시작 시 스폰)

그레이스 선택 처치 (권장)

그레이스 청크는 완전 선택. 동 윙 (2층)으로 가서 Wristband ID (Level 1) 위치 복도. 느리게 움직이므로 뒤에서 매복 가능.

// 헤모리틱 인젝터 방법 (권장)

그레이스가 레퀴엠 혈액 분석기 퍼즐 풀고 나면 헤모리틱 인젝터 제작 가능. 3개 인젝터 = 즉사 — 뒤에서 몰래 접근 후 빠르게 트리거.

이게 자원 효율적입니다. 3개 인젝터 제작 < 2-3 레퀴엠 탄환, 그리고 레퀴엠은 후반에 더 희귀.

// 왜 그레이스가 처치해야

그레이스로 처치 시 Grace and Goliath 도전 완료 → 독특한 부적 보상. 또한 레온의 다락방 조우가 선택적으로 변경 — Platinum 런에서 8-12분 절약.

레온 필수 다락방 전투

그레이스가 청크를 죽이지 않으면 레온이 다락방에서 직면. 전형적 RE 보스 아레나 — 좁은 방, 사다리 수직 이동, 제한된 보급.

// 1단계 — 샷건 즉시 픽업

조우 시작 순간 입구 근처 바닥의 MSBG 500 샷건으로 스프린트. 가장 중요한 행동 — 핸드건 데미지는 청크 덩치에 무의미.

// 2단계 — 머리/배 타격

주로 청크 머리, 정면 때 배가 보조. 점블랭크 샷건이 큰 데미지. 사격 사이 이동 — 가만히 서면 그랩 당함.

// 3단계 — 노란 사다리로 상층

다락방의 노란 사다리로 상층 플랫폼 접근. 등반 후 청크가 따라오기 시도할 때까지 대기 → 머리와 손에 샷건 발사. 그를 스태거시키고 하층으로 떨어뜨림 → 잠깐 스턴.

// 4단계 — 피니셔

하층 스턴 상태에서 빠르게 드롭 → 머리로 스프린트 → 근접 피니셔 프롬프트. 큰 추가 데미지.

// 종료 신호

청크가 몸 전체 구멍에서 피를 뿜기 시작하면 전투 사실상 종료. 계속 쏘지 마세요 — 후퇴 후 자체 폭발 대기. 폭발 시 너무 가까이 있으면 체력 감소, 방 가장자리로.

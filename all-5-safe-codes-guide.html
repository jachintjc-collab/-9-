Every safe combination in Resident Evil Requiem (RE9) for both Normal and Insanity difficulty. Opening all 5 safes unlocks the "No Safe Is Safe" trophy and rewards crucial Antique Coins, ammo, and healing items. Only Grace can open safes — Leon cannot.

⚠ Important: Insanity difficulty uses completely different combinations than Normal difficulty. Don't try Normal codes on Insanity — they won't work. Direction notation: L = Left (clockwise), R = Right (counterclockwise).

Safe #1 — Bar & Lounge // CARE CENTER WEST WING

The first safe most players will encounter, located in the West Wing of the Care Center. Easy to miss because it's hidden behind the bar counter.

// SAFE #1

Bar & Lounge Safe

Normal / Standard difficulty
L10 · R80 · L30
Insanity difficulty
L40 · R70 · L20
Location: Behind the bar counter, to the right of the flower arrangement. Clear out the zombies in the Bar & Lounge first — they will interrupt the dial input.
Code clue: Found in the West Wing Office on a clipboard near the desk.
Reward: 3× Antique Coins (spend at the Parlor)

Safe #2 — Examination Room // CARE CENTER EAST WING 1F

Located in the Examination Room on the East Wing's first floor. Tucked into a corner cabinet that's easy to walk past.

// SAFE #2

Examination Room Safe

Normal / Standard difficulty
R30 · L10 · R50
Insanity difficulty
R60 · L20 · R40
Location: Far corner of the Examination Room, inside a metal cabinet on the floor. The cabinet has visible scratches.
Code clue: On East Wing 2F, in the same room where you encounter the giant fat zombie (Chunk). Look on the medical chart.
Reward: 3× Antique Coins

Safe #3 — Furnace Room // CARE CENTER BASEMENT

One of the harder safes to access — requires the Level 1 Wristband from solving the first Quartz Puzzle Box on East Wing 2F. Don't try to reach this safe before completing that puzzle.

// SAFE #3

Furnace Room Safe (Basement)

Normal / Standard difficulty
L60 · R20 · L40
Insanity difficulty
L30 · R50 · L70
Location: Inside the Furnace Room in the Care Center Basement. Requires Level 1 ID Wristband to access the basement security door.
Code clue: File found in the Boiler Room near the maintenance schedule.
Reward: 10× Shotgun Shells + 2× Antique Coins
💡 Bonus tip: The Workshop in the basement has a separate safe (R60·L40·R80) that contains the Raccoon Roundup map. This map highlights all Care Center Mr. Raccoon Memoriams on your map. Highly recommended for 100% runs.

Safe #4 — Sterilization Chamber // CARE CENTER STERILIZATION

This safe contains some of the best mid-game rewards in the Care Center, including Hemolytic Injectors that one-shot mini-bosses like Chunk and the Chef.

// SAFE #4

Sterilization Chamber Safe

Normal / Standard difficulty
R70 · L50 · R10
Insanity difficulty
R20 · L80 · R60
Location: Anteroom near the Bioweapon Repository. The safe is partially hidden behind a wooden crate — push it aside or shoot it apart.
Code clue: On a research note inside the Sterilization Chamber's clean zone.
Reward: 2× Hemolytic Injector + 1× Antique Coin

Safe #5 — Monitor Control Room // RACCOON CITY

The final safe is in Raccoon City, far from the Care Center. Highly missable — you can't return here once you progress past the Bioweapon Repository sequence.

// SAFE #5 — FINAL SAFE

Monitor Control Room Safe

Normal / Standard difficulty
L90 · R30 · L60
Insanity difficulty
L20 · R40 · L80
Location: Bioweapon Repository 05, rightmost open container. Look for the metal locker labeled "Monitor Control."
Code clue: Found in the Records Room on a security memo.
Reward: 5× Requiem Bullets + High-Grade Herb

Tips, tricks, and trophy reward

// HOW TO INPUT SAFE CODES

Approach the safe and interact with it. The dial puzzle screen appears. Use these inputs:

  • L (Left) = rotate dial clockwise
  • R (Right) = rotate dial counterclockwise
  • Stop precisely on the listed number — overshooting does not work, you must exit and retry
  • Press the Check / Confirm button after entering all three numbers

// WHY ONLY GRACE CAN OPEN SAFES

Safe-cracking is one of Grace Ashcroft's character-specific abilities, like Blood Crafting. Leon cannot interact with safes at all — he won't even see the prompt. If you're playing as Leon and trying to open a safe, switch back to Grace's section in the story or use a Grace-only save.

// "NO SAFE IS SAFE" TROPHY

Opening all 5 safes (regardless of difficulty) unlocks the "No Safe Is Safe" Silver trophy. This is one of the easier collectible-based trophies in the game if you have this guide handy. Combine it with the Mr. Raccoon collectible run for an efficient cleanup pass.

// WHAT TO DO WITH ANTIQUE COINS

The 12+ Antique Coins from safes should be spent at The Parlor in the West Wing. The Parlor merchant offers:

  • Weapon attachments (scopes, extended mags, suppressors)
  • Healing items at discounted rates
  • Crafting materials for Grace's Blood System
  • Rare cosmetics for post-game

Don't hoard coins — there are more than enough in the game to fully stock up at the Parlor.

Speedrun strategy: Memorize codes, skip clues

For Insanity speedrun attempts, memorizing all 5 codes from this guide saves significant time. The code clues are scattered across optional rooms — skipping them and entering codes directly cuts roughly 8-10 minutes off a typical Care Center run.

Recommended speedrun order:

  1. Safe #1 (Bar & Lounge) — accessible in early Care Center exploration
  2. Safe #2 (Examination Room) — same area, same trip
  3. Safe #3 (Furnace Room) — after getting Level 1 Wristband
  4. Safe #4 (Sterilization Chamber) — same area as Safe #3
  5. Safe #5 (Monitor Control Room) — Raccoon City, last opportunity

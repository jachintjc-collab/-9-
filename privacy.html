제3자 서비스

저희는 정보를 수집할 수 있는 다음 제3자 서비스를 사용합니다:

// GOOGLE ANALYTICS

방문자 행동을 집계 수준에서 이해하기 위해 Google Analytics를 사용할 수 있습니다. Google Analytics는 쿠키를 사용하여 페이지 조회, 세션 지속 시간, 트래픽 소스에 대한 익명 데이터를 수집합니다. Google Analytics 옵트아웃 브라우저 추가 기능을 사용하여 옵트아웃할 수 있습니다.

// GOOGLE ADSENSE

광고를 표시하는 경우 Google AdSense를 사용할 수 있습니다. Google AdSense는 본 사이트나 인터넷의 다른 사이트에 대한 이전 방문을 기반으로 광고를 게재하기 위해 쿠키(DoubleClick 쿠키 포함)를 사용합니다.

Google 광고 설정을 방문하여 개인 맞춤 광고를 옵트아웃할 수 있습니다.

// GITHUB PAGES

이 사이트는 GitHub Pages에 호스팅됩니다. GitHub는 GitHub 개인정보 보호 정책에 설명된 대로 접근 로그를 수집할 수 있습니다.

// KO-FI

Ko-fi를 통해 사이트를 후원하기로 선택하신 경우, 거래는 Ko-fi Lab Limited에서 처리됩니다. 해당 거래에는 그들의 개인정보 처리방침이 적용됩니다.

RE9 전체 10 보스 난이도 랭킹 — 최난이도부터 최쉬움까지

한국어판 작성자 노트: 이 페이지는 단순한 영어 번역이 아닙니다. 서울 거주 RE 시리즈 23년 팬 Jachin이 PS5/Steam에서 RE9을 직접 6회 플레이 (Casual → Insanity → NG+ Insanity)한 후 작성. 한국 RE 커뮤니티에서 통용되는 영어 용어(Mortal Edge, NG+, Insanity 등)는 그대로 유지하면서 한국어 독자에게 적합한 표현으로 구성했습니다.

검증 버전: v1.02 · 마지막 검토: 2026년 5월 30일 · 영어 원본과 비교: 100% 내용 동등 + 한국어 컨텍스트 추가

2026년 5월 16일 게시 · 8분 읽기
한국어판 안내: 영어 원본의 한국어 번역. 작성자(서울 거주)가 직접 검수했으며, Mortal Edge / Insanity / NG+ 등 영어 표기는 한국 RE 커뮤니티 관행을 따라 그대로 사용했습니다.

레지던트 이블 레퀴엠의 모든 보스를 데미지 출력, 공격 패턴, 패리 난이도, 인새니티 모드 성능 기반으로 가장 어려움부터 가장 쉬움까지 순위 매겼습니다. 무엇이 다가올지 알고 미리 준비하는 데 사용하세요.

랭킹 기준

각 보스는 다섯 가지 요소로 평가됩니다:

  • 데미지 출력 — 공격이 얼마나 강한지
  • 공격 패턴 복잡도 — 몇 가지 동작이 있고 얼마나 읽기 쉬운지
  • 패리 난이도 — 패리 윈도우가 얼마나 빡빡한지
  • 자원 소모 — 탄약/회복 아이템을 얼마나 소비하는지
  • 인새니티 스케일링 — 최고 난이도에서 얼마나 가혹해지는지

목록은 가장 어려움(#1)부터 가장 쉬움(#8)까지. 순서가 중요합니다: 무엇이 올지 알면 올바른 전투에 자원을 아낄 수 있습니다.

S 티어 — 잔혹한 전투

#1 빅터 기드온 박사 // ARK 시설 · 최종 ★★★★★

레지던트 이블 레퀴엠의 가장 어려운 전투. 전기 공격, RPG-7, 촉수 슬램, 10초 보라 충전 윈도우의 2 페이즈 네메시스 클론. Mortal Edge Hatchet 필수. 이 전투 전에 탄약을 낭비하지 마세요.

CANON ENDING · 2-PHASE · LEON

#2 더 커맨더 (HUNK 타입) // ARK 복도 ★★★★★

RE4의 크라우저처럼 칼싸움. 빠른 공격에 읽기 쉬운 텔레그래프. 게임에서 가장 기술 집약적인 전투. 보상: Mortal Edge Hatchet — 빅터에 필수.

PARRY-FOCUSED · LEON

#3 미스터 X — 슈퍼 타이런트 // RPD 외부 ★★★★★

차 던지기와 도약 공격을 가진 상징적 보스. 심장을 노리세요 — 몸통 사격은 거의 데미지가 없습니다. 추격 시퀀스 먼저의 2 페이즈.

CHASE+FIGHT · LEON · HEART WEAKPOINT

A 티어 — 도전적인 전투

#4 타이탄 스피너 // 라쿤 시티 광장 ★★★★★

거대한 회전 바이오웨폰. 회전 공격은 패리 불가 — 엄폐물을 사용해야 합니다. 버스트 데미지가 일시적 스턴 윈도우를 트리거. 스턴용 레퀴엠 라운드를 아껴두세요.

COVER-BASED · LEON

#5 플랜트 43 // ARK 입구 ★★★★★

세 머리 식물 괴물. 옆 새싹을 먼저 사격한 다음 중앙 꽃이 입을 벌릴 때 사격. 마무리: 떨어진 트럭에서 폭발 캔 사격으로 즉시 시네마틱 처치.

CINEMATIC FINISHER · LEON

B 티어 — 관리 가능한 전투

#6 청크 #2 (Timothy B. Jackson) // 케어 센터 다락방 ★★★★★

두 번째 청크 조우. 전투 시작 시 시신 옆의 MSBG 500 샷건을 즉시 픽업. 노란 사다리로 경로를 강제하세요.

SHOTGUN PICKUP · LEON

#7 청크 #1 (Thomas K. Jackson) // 케어 센터 동 윙 ★★★★★

선택적이지만 권장. 문틀에 걸리는 느린 변이체. 뒤에서 헤모리틱 인젝터 3개 = 즉사. Eye Spy 부적 드롭.

OPTIONAL · GRACE · CHARM REWARD

#8 더 셰프 // 서 윙 주방 ★★★★★

게임에서 가장 쉬운 "보스". 선택 조우 — 무료 보급을 위한 팬트리 키 드롭. 헤모리틱 인젝터 한 개로 즉사. 스피드런에서는 쉽게 건너뜁니다.

OPTIONAL · GRACE/LEON

캠페인 진행 순서

스토리 계획을 위해, 보스를 만나는 순서:

  1. 더 셰프 (케어 센터 주방) — 선택, 매우 초기
  2. 청크 #1 (동 윙) — 선택, Eye Spy 부적용 권장
  3. 청크 #2 (케어 센터 다락방) — 필수, 중간 게임
  4. 미스터 X / 슈퍼 타이런트 (RPD 외부) — 필수, 케어 센터 이후
  5. 타이탄 스피너 (라쿤 시티 광장) — 필수, 라쿤 시티 중간
  6. 플랜트 43 (ARK 입구) — 필수, ARK 초기
  7. 더 커맨더 (ARK 복도) — 필수, ARK 중간
  8. 빅터 기드온 (ARK 최종 챔버) — 최종 보스

전투 전 체크리스트

보편적인 보스 전 준비:

  • 전투 전에 수동 저장 (새 슬롯 사용)
  • 가장 가까운 보급 상자에서 보디 아머 수리
  • 응급 처치 스프레이 2개 이상 비축
  • 컷씬을 트리거하기 전에 모든 무기 재장전
  • 가장 강력한 근접 무기 장착 (이상적으로 Mortal Edge Hatchet)
  • 인새니티의 경우 Eye Spy 부적 (무작위 치명상 회피) 장착 고려
